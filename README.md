# Risk2Safe RA Studio
### AI-Powered Task-Based Risk Assessment

A world-class, standalone risk assessment application built on:
- **EI 3580** — Guidance on Human Factors in Task-Based Risk Assessment (Energy Institute, 2025)
- **ADNOC Corporate 6×6 Risk Matrix** — Initial and residual risk ranking
- **Anthropic Claude AI** — Hazard, EPC and control suggestions

## Features
- 6-stage EI 3580 methodology workflow
- 10 energy sources → hazard identification
- Error-Producing Conditions (PIFs): Job / Person / Organisation
- Hierarchy of controls (Eliminate → Substitute → Engineering → Admin → PPE)
- Interactive ADNOC 6×6 risk matrix (CAT1–CAT4)
- 9 Frontline Tools (pre/during/post job)
- Excel, Word and JSON export
- AI assistance via Anthropic Claude (bring your own API key)
